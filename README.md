# NSD-Nusantara-Spirit-Dono-
NSD (Nusantara Spiritual Donation) adalah platform donasi darurat yang menghubungkan donatur, pemohon bantuan, konselor, dan admin dalam satu ekosistem terpercaya. Dilengkapi verifikasi pengajuan, pelacakan donasi real-time, transparansi dana, serta dukungan konseling untuk memastikan bantuan tepat sasaran
